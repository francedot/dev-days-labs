Xamarin Dev Days - Torino

  • 08/06/2017
  • Torino, Italy

 

 

Il Microsoft Innovation Center Torino in collaborazione con I3P, Incubatore Politecnico di Torino, le Community Torino .NET User Group e TTG , Xamarin e Microsoft, organizza l’8 giugno 2017 a Torino una delle tappe italiane dello Xamarin Dev Day.

Un’intera giornata GRATUITA guidata dagli MVP e dagli esperti Xamarin per esplorare lo sviluppo di applicazioni mobile attraverso sessioni teoriche dedicate a Xamarin, a Xamarin.Forms e agli Azure Services. E nel pomeriggio un workshop pratico per imparare a sviluppare un’app Xamarin connessa al cloud Microsoft Azure con il supporto degli esperti.

Come prepararsi:

  • Porta il tuo PC o Mac con Xamarin già installato e configurato PRIMA DELL’EVENTO:
    • Su Windows: Visual Studio 2015/2017 con integrazione Xamarin
    • Su Mac: Xamarin Studio o Visual Studio for Mac Preview

Per connettere la propria app con Azure Mobile App Service è necessario un piano Azure. Se non ne possiedi già uno, ti forniremo un Azure Free Pass durante l’evento.

  • Materiale Utile:
    • Installazione Online
    • Installazione Offline
    • Azure
    • Troubleshooting
    • Xamarin in VS2017
    • Creare la prima App: File -> New App

E quindi cosa aspetti? Ti aspettiamo allo Xamarin Dev Day Torino!
